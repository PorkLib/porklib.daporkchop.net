**PorkLib** is a collection of Java libraries made by [DaPorkchp_](http://daporkchop.net) designed to make common tasks less repetitive.  

## Libraries

- [crypto](https://github.com/PorkLib/crypto)
- [encoding](https://github.com/PorkLib/encoding)
- [hash](https://github.com/PorkLib/hash)
- [http](https://github.com/PorkLib/http)
- [math](https://github.com/PorkLib/math)
- [nbt](https://github.com/PorkLib/nbt)
- [network](https://github.com/PorkLib/network)
